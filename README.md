owoc nauki linuxa

Simple bash healthcheck utility that displays:
-User
-Hostname
-Kernel version
-Uptime
-Disk usage
-RAM usage
-Realtime internet connection test
-Number of errors in journalctl since boot
-Statuses of NetworkManager, Bluetooth, Cups, sshd, using systemctl.

technologies used:
-bash
-awk
-systemctl
-journalctl
-ping
